  * **发往国内号码**：格式为 `+86185xxxxxxxx` 或 `0086185xxxxxxxx`。

#### 2. 短信发送失败/发不出的解决办法
如果你在发送短信时遇到感叹号或发送失败提示，请依次尝试以下通用修复流程：
1. 手机开启 **飞行模式** 并在原地等待 **15 秒**，随后关闭飞行模式重试。
2. 若依旧不行，进入手机设置：`设置` -> `移动网络` -> `SIM卡管理` -> 选择 `giffgaff`。
3. **关闭“自动选择网络”** 开关，让手机重新搜索周围的所有运营商信号。
4. 在弹出的信号列表中，**手动选择“中国移动”线路**（如果移动不行，再依次尝试手动切换至中国联通或其他可用运营商）。

4. 在列表中 **手动选择“中国移动”**（若移动不行，可尝试手动切换至其他可用运营商）。

### 2. 特定 App 无法收到验证码的解决方案
如果你已经有了信号，但某些特定应用（如 Telegram、X、WhatsApp）仍收不到验证码，请尝试以下方法：

* **重置网络线路**：先按照上面的步骤手动锁定到 **中国移动** 线路（若不行再切换其他运营商）。
* **X (推特) / WhatsApp**：
  * 请确保网络代理（梯子）开启 **全局模式**。
  * 建议更换为 **英国 (UK)** 或 **美国 (US)** 较干净、未被污染的节点再试。
* **Telegram (电报)**：
  * **安卓端**：请务必从 Google Play 商店下载官方的 **Telegram X** 客户端进行注册/登录。
  * **苹果端**：请前往 App Store（需切换至美区等海外 Apple ID）下载官方客户端。
  * **⚠️ 报错提示“验证码已发送至其他设备”的解决办法**：
    如果你在注册或登录时，系统提示 *“We've sent the code to the Telegram app on your other device”*（验证码已经发到其他 Telegram App），但你实际上根本没有登录过其他设备，请尝试以下“强制触发短信”的办法：
    1. 找一个**已经有 Telegram 账号的朋友**（或者用你自己的国内号/其他老账号）。
    2. 让他在他的手机端 Telegram 上，点击 **Contacts（联系人）** -> 点击 **「+」号**。
    3. 名字随便填，手机号输入你这张**未注册成功的 Giffgaff 号码**，然后点击 **Create（创建）**。
    4. 由于你的号码还没注册过，此时他的 Telegram 客户端就会强制向你的 Giffgaff 手机号**发送一条邀请注册的系统短信**。
    5. 收到这条短信后，你再回到自己的手机上重新走一遍注册流程，通常就能正常接收到验证码了。
